# СОВФД — информационный сайт

Многостраничная статическая версия сайта ОГАУЗ «Смоленский областной врачебно-физкультурный диспансер».

Содержит главную страницу, сведения об организации, информацию для пациентов, список медицинских работников, контакты, новости и отделения. Контактные данные и материалы перенесены с официального сайта `sovfd67.gosuslugi.ru`.

## Локальный запуск

```powershell
py -3 -m http.server 4173 --bind 127.0.0.1
```

После запуска откройте `http://127.0.0.1:4173/`.
