# CYBERTAG Lite

Статическая версия сайта CYBERTAG с новой компоновкой, сохраненными логотипами, данными оборудования, комплектов, акций и контактов.

## Локальный запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```powershell
python -m http.server 4173
```

## Демо-вход

- Пользователь: `manager`
- Пароль: `cybertag2026`

Вход, корзина и заявки работают в `localStorage`, поэтому данные сохраняются только в текущем браузере. Для настоящей авторизации нужен backend или внешний auth-сервис.

## GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этой папки в репозиторий.
3. В настройках репозитория откройте `Pages`.
4. Выберите публикацию из ветки `main`, папка `/root`.

После сохранения сайт будет доступен по адресу GitHub Pages.
