# СГУС — навигатор абитуриента

Многостраничный информационный сайт Смоленского государственного университета спорта.

Разделы: информация о вузе, девять программ обучения, специальности с отдельными подробными страницами, факультеты, контакты приёмной комиссии, калькулятор среднего балла и административная панель.

## Панель управления

Адрес: `/admin/login/`

- логин: `admin`
- пароль: `SGUS2026`

Админ-панель позволяет добавлять, редактировать и удалять факультеты и специальности, а также экспортировать и импортировать каталог в JSON. Поскольку сайт работает на GitHub Pages без серверной базы данных, изменения сохраняются в `localStorage` текущего браузера.

## Локальный запуск

```powershell
py -3 -m http.server 4173 --bind 127.0.0.1
```
