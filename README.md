# СГУС — навигатор абитуриента

Многостраничный информационный сайт Смоленского государственного университета спорта.

Разделы: информация о вузе, девять программ обучения, семь специальностей, два факультета, контакты приёмной комиссии и калькулятор среднего балла.

## Локальный запуск

```powershell
py -3 -m http.server 4173 --bind 127.0.0.1
```
