# CYBERTAG Lite

Статическая многостраничная версия сайта CYBERTAG с портальной компоновкой по образцу из технического задания, чёрно-красной палитрой и сохранёнными логотипами и данными.

## Локальный запуск

Запустите локальный сервер из корня проекта:

```powershell
py -3 -m http.server 4173 --bind 127.0.0.1
```

Страницы: `/`, `/katalog/`, `/bundles/`, `/soft/`, `/akcii/`, `/kontakty/`, `/korzina/`, `/login/`.

## Демо-вход

- Пользователь: `manager`
- Пароль: `cybertag2026`

Кнопка оформления заказа переводит неавторизованного пользователя на отдельную страницу входа. Вход, корзина и заявки работают в `localStorage`, поэтому данные сохраняются только в текущем браузере. Для настоящей авторизации нужен backend или внешний auth-сервис.

## GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этой папки в репозиторий.
3. В настройках репозитория откройте `Pages`.
4. Выберите публикацию из ветки `main`, папка `/root`.

После сохранения сайт будет доступен по адресу GitHub Pages.
